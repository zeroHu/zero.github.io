一点小思考

那我们可以从这个例子里面去分析哈，首先 accessToken 的获取需要我方的 appKey 与 appSecret 一起获取，那么可以保证这个是不被篡改的。
那么我们今天就是研究一下这里的 code 是如何生成的，是什么机制然后如何能保证 5 分钟过期，且调用一次过期。做到与 accessToken 一起使用，发送给微信，保障了信息获取流程的安全性。
