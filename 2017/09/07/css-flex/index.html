栅格系统特性

  • 每一行的每一个栅格默认都是同宽通告，默认自适应
  • 为了足够灵活，能够添加尺寸属性到单独的栅格中。没有添加的，仍然简单的平分剩下的可用空间
  • 支持响应式布局，可以添加媒体查询到栅格中
  • 每一个栅格可以在垂直方向上置顶，置底，居中
  • 栅格可以随意嵌套
